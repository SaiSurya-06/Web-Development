HTML - LINKS

Creating a web link

- Use <a> tag to create a hyperlink
- Use href attribute to specify the URL of the page the link goes to
Ex: <a href= "https://www.google.com"> Google

Creating links within same web

- Use <a> tag to create a hyperlink
- Use href attribute to specify the Section of the page the link goes to
Ex: <a href= "#introduction"> Introduction

HTML - IMAGE

- Use alt attribute to specify an alternate text for the image
Ex: <img src="image.jpg" alt="Image">

Resizing an image

- Use width attribute to specify the width of the image
- Use height attribute to specify the height of the image
Ex: <img src="image.jpg" alt="Image" width="100" height="100">

HTML - ATTRIBUTES

- Attributes are used to provide additional information about an HTML element
- Attributes are specified in the opening tag of an HTML element
- Attributes are separated by an equal sign (=)
- Attributes have a name and a value
Ex: <input type="text" value="Name">
- Common attributes include id, class, src, href, alt, title, width, height, rows, cols, name, value, for, required, checked, disabled, readonly, multiple, accept, pattern, min, max, step
Ex: <input type="text" id="name" value="Name">

HTML - META ELEMENTS

- Use <meta> tag to provide metadata about an HTML document
- Use charset attribute to specify the character encoding for the document
- Use name attribute to specify the name of the metadata
- Use content attribute to specify the value of the metadata
Ex: <meta charset="UTF-8">
- Common metadata include description, keywords, author, viewport
Ex: <meta name="description" content="Free Web tutorials">

HTML - ESCAPE CHARACTERS

- Use & lt; to display a less than sign (<)

- Use & nbsp; to display a non-breaking space

HTML - AUDIO ELEMENT

- Use <audio> tag to embed audio content in a document
- Use src attribute to specify the URL of the audio file
- Use controls attribute to display audio controls
Ex: <audio src="audio.mp3" controls></audio>

- Use autoplay attribute to automatically start playing the audio
Ex: <audio src="audio.mp3" controls autoplay></audio>
- Use loop attribute to automatically start playing the audio
Ex: <audio src="audio.mp3" controls autoplay loop></audio>

HTML - VIDEO ELEMENT

- Use <video> tag to embed video content in a document
- Use src attribute to specify the URL of the video file
- Use controls attribute to display video controls
Ex: <video src="video.mp4" controls></video>

- Use autoplay attribute to automatically start playing the video
Ex: <video src="video.mp4" controls autoplay></video>
- Use loop attribute to automatically start playing the video
Ex: <video src="video.mp4" controls autoplay loop></video>
- Use width attribute to specify the width of the video
- Use height attribute to specify the height of the video
Ex: <video src="video.mp4" controls autoplay loop width="320" height="240"></video>
